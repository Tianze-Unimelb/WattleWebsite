# WattleWebsite
## Melbourne Sustainable Start - website prototype

This folder contains a static one-page website for the Wattle Fellowship optional submission.

### Files

- `index.html` - the website page
- `styles.css` - all styling, no external dependencies
- `assets/favicon.svg` - small decorative icon
- `Melbourne_Sustainable_Start01.pdf` - one-page project mock-up linked from the website

### Note

This is an early prototype created for a fellowship application. It is not an official University of Melbourne resource.
