# WattleWebsite
## Melbourne Sustainable Start - website prototype

This folder contains a static one-page website for the Wattle Fellowship optional submission.

### Files

- `index.html` - the website page
- `styles.css` - all styling, no external dependencies
- `assets/favicon.svg` - small decorative icon
- `Melbourne_Sustainable_Start_Project_Mockup.pdf` - one-page project mock-up linked from the website

### Suggested submission workflow

1. Upload this folder to a static hosting service such as Netlify, GitHub Pages, Google Sites, or Canva website.
2. Confirm the website link opens without sign-in.
3. Confirm the PDF download link works.
4. Put the website URL in the application form's link field.
5. Upload the standalone PDF separately in the file upload field if the form allows it.

### Note

This is an early prototype created for a fellowship application. It is not an official University of Melbourne resource.
